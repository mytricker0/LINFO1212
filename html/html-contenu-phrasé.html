Cette page illustre quelques éléments de contenu phrasé en HTML.

Sémantique du texte

Le texte peut être insistant, important ou pertinant.

Le texte peut être italique ou gras (sans sémantique prédéfinie, et donc plutôt à éviter).

Le texte peut être des petits caractères concernant par exemple une information légale.

Le texte peut être en superscript ou subscript, par exemple pour les ordinaux: 1er, 2ème, 3ème, …

Le texte peut être biffé.

Le texte peut être signalé comme étant éliminé ou inséré.

Le texte peut être l'entrée ou la sortie d'un programme, ou un bout de code source.

Bien que certains des éléments illustrés précédemment ont la même apparence par défaut, le plus important est la semantique que l'on donne au texte avec chaque élément. On peut utiliser CSS pour faire une différentiation visuelle des éléments.
